# JurigestEnterprise
JuriGest Enterprise es una plataforma integral para la gestión de oficinas de Receptores Judiciales de Chile.  El sistema ha sido diseñado utilizando principios de Domain-Driven Design (DDD), Clean Architecture y .NET 9, con el objetivo de proporcionar una solución moderna, escalable y mantenible para la administración completa de causas judiciales
